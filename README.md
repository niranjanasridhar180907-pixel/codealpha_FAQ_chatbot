# 🤖 AI FAQ Chatbot

## 📌 Project Overview

The AI FAQ Chatbot is an intelligent question-answering application developed using Python, Streamlit, and Natural Language Processing (NLP). The chatbot helps users find answers to frequently asked questions by analyzing user queries and matching them with the most relevant information from a predefined knowledge base.

This project was developed as part of the **CodeAlpha Artificial Intelligence Internship Program**.

## ⚙️ Working Principle

1. User enters a question.
2. The question is preprocessed using NLP techniques.
3. TF-IDF Vectorization converts text into numerical features.
4. Cosine Similarity identifies the most relevant FAQ.
5. The chatbot returns the best matching answer along with a confidence score.
6. Chat history is stored and can be downloaded.

---

## 📊 Key Concepts Used

### Natural Language Processing (NLP)

Natural Language Processing enables computers to understand and process human language.

### TF-IDF Vectorization

TF-IDF (Term Frequency-Inverse Document Frequency) converts text into numerical vectors for similarity comparison.

### Cosine Similarity

Cosine Similarity measures the similarity between user queries and stored FAQ questions to identify the best match.
